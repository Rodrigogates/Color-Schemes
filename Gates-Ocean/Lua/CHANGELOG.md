## [1.0.0] - 2024-02-09

### Added
- Initial release of Gates Ocean colorscheme
- Support for TreeSitter highlighting
- Support for LSP diagnostics
- Terminal color integration
- Custom highlights for popular plugins:
  - Telescope
  - NvimTree / Neo-tree
  - Git Signs
  - Which-key
- Configurable options:
  - Transparency support
  - Customizable styles (italic, bold, etc.)
  - Terminal colors toggle
- Installation script for easy setup
- Comprehensive documentation in README
- Example configurations for LazyVim and standard Neovim

### Color Palette
- Deep ocean blue-green theme inspired by underwater depths
- Carefully selected colors for optimal readability and reduced eye strain
- Based on the Gates-Ocean theme for Notepad++
